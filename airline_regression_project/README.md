# Flight Price Regression Project

Isi folder:
- data/airlines_flights_data.csv: dataset yang digunakan.
- notebook/Flight_Price_Regression_EDA_Modeling.ipynb: code EDA, preprocessing, training, dan evaluation untuk Google Colab.
- app/app.py: source code Streamlit untuk deployment model.
- model/flight_price_model.pkl: model terlatih.
- slides/Flight_Price_Regression_Presentation.pptx: slide presentasi.
- LINK_GOOGLE_COLAB.txt: catatan link Colab.

Cara menjalankan Streamlit:
1. Buka terminal di folder app.
2. Install dependency: pip install -r requirements.txt
3. Jalankan: streamlit run app.py
